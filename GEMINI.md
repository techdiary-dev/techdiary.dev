## Project Overview

Techdiary.dev is a platform for developers to share their knowledge and experiences.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Drizzle ORM
- Cloudflare Workers

## Getting Started

To get started with the project, run the following commands:

```bash
npm install
npm run dev
```

## Testing

To run the tests, use the following command:

```bash
npm run test
```
